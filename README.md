# RGB-Bear-Lamp

This project is a low poly bear-shaped decorative lamp made from paper, designed to integrate an internal electronic system using an ESP32 and addressable RGB LEDs. The lamp combines aesthetic design with embedded electronics to create a soft, customizable lighting experience. To learn how to combine electronics and design into a functional and creative project.
